# MyHeritage 功能特性分析

## 概述
MyHeritage 是一家以色列的基因和家谱公司，成立于2003年。它提供全面的家谱服务，包括在线家谱树构建、历史记录搜索、DNA测试等。MyHeritage 致力于帮助用户发现和研究家族历史，连接全球用户与他们的祖先和亲属。

## 核心功能

### 1. 家谱构建 (Family Tree Building)
- **在线家谱树**：用户可以通过简单的输入开始构建家谱
- **自动匹配技术**：系统自动匹配帮助用户绘制家谱图
- **Smart Matches™**：智能匹配技术连接用户与相关家谱
- **Record Matches**：将用户家谱与历史记录关联
- **协作功能**：支持家庭成员共同编辑和维护家谱

### 2. 历史记录搜索 (Historical Records Search)
- **庞大数据库**：拥有超过35.1亿个历史记录
- **全球覆盖**：记录涵盖66个国家的历史文档
- **多样化记录**：包括出生、死亡、婚姻、人口普查、移民等记录
- **多语言支持**：支持42种语言的记录搜索
- **持续增长**：每年增加数十亿条记录

### 3. DNA测试服务 (DNA Testing Service)
- **族裔分析**：通过脸颊拭子测试了解用户的族裔血统
- **亲属匹配**：通过安全的DNA数据库连接亲属
- **地理区域分析**：提供详细的地理区域来源分析
- **特征分析**：分析受基因影响的75多种特征
- **家庭关系确认**：帮助确认家庭关系和未知亲属

### 4. 照片增强功能 (Photo Enhancement Features)
- **照片修复**：修复老旧和损坏的家庭照片
- **照片着色**：将黑白照片转化为彩色照片
- **Deep Nostalgia™**：利用AI技术让历史照片中的人物动起来
- **LiveMemory™**：将照片转化为动画视频
- **Photo Storyteller™**：为照片添加故事和背景信息

### 5. 移动应用功能
- **家谱研究**：随时随地进行家谱研究
- **照片上传**：方便上传和管理家庭照片
- **即时发现**：Instant Discoveries™功能一键添加分支和照片
- **离线访问**：支持离线查看家谱信息
- **社交分享**：与家人和朋友分享发现
